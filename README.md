# Nebula Craft - Site Oficial do Servidor

## Descrição
Este projeto é a landing page oficial do servidor fictício de Minecraft "Nebula Craft". O site foi desenvolvido como avaliação da disciplina de Linguagem de Marcação e Formatação, utilizando um tema "Dark/Gaming" para apresentar os modos de jogo e área de suporte.

## Tecnologias Utilizadas
- HTML5 (Estrutura Semântica)
- CSS3 (Estilização Dark Mode e Fontes Monospace)
- Bootstrap 5 (Grid Responsivo, Navbar, Cards, Alerts, Badges)
